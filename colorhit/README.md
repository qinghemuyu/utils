# ColorHit 自动点击工具

## 项目简介

ColorHit 是一个基于颜色识别的自动点击工具。它能够监控屏幕指定区域，当检测到目标颜色时自动执行点击操作。这个工具特别适用于需要快速响应颜色变化的场景。

## 功能特点

- **实时颜色检测**：持续监控屏幕中心区域的颜色变化
- **智能取色**：支持通过鼠标点击进行屏幕取色
- **自动点击**：当检测到目标颜色时自动执行点击操作
- **可配置参数**：支持自定义检测区域大小和颜色阈值
- **实时颜色预览**：取色过程中实时显示鼠标位置的RGB值

## 安装说明

1. 确保已安装Python环境
2. 安装所需依赖：
   ```bash
   pip install -r requirements.txt
   ```
3. 直接运行可执行文件 `ColorHit.exe`，或通过Python运行源代码：
   ```bash
   python color_detector.py
   ```

## 使用方法

1. 启动程序后，会有3秒的准备时间，可以切换到目标窗口
2. 程序运行时：
   - 按 `P` 键进入取色模式
   - 移动鼠标查看实时RGB值
   - 点击鼠标选择目标颜色
   - 按 `Ctrl+C` 停止程序

## 配置说明

程序使用 `color_config.json` 存储配置信息，包含以下参数：

```json
{
    "r_threshold": 150,    // 红色分量阈值
    "g_threshold": 100,    // 绿色分量阈值
    "b_threshold": 100,    // 蓝色分量阈值
    "area_size": 30,      // 检测区域大小（以屏幕中心为原点的方形区域边长的一半）
    "step": 2             // 检测步长（每隔多少像素取样一次）
}
```

- 颜色阈值：通过取色功能自动更新
- 检测区域：可以通过修改配置文件调整大小
- 检测步长：值越小检测越精确，但会增加CPU负载

## 注意事项

1. 首次运行时会自动创建默认配置文件
2. 取色时请确保鼠标在屏幕范围内
3. 程序会在屏幕中心区域进行颜色检测
4. 检测到目标颜色时会自动点击屏幕中心位置
